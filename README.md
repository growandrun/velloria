# Velloria — 실버 테니스 팔찌 랜딩페이지

> **"손목 위 한 줄의 빛"** · 큐빅 지르코니아(CZ) 실버 테니스 팔찌 브랜드의 단일 페이지 랜딩.
> 구매는 **eBay 리스팅에서 완료**됩니다 — 이 페이지는 제품을 보여주고 eBay로 연결하는 역할을 합니다.

- **라이브:** https://velloria.vercel.app
- **언어:** 한국어
- **구성:** 정적 HTML **단일 파일** — 빌드 단계 없음, 외부 의존성 없음 (폰트·제품 사진이 base64로 내장됨)

---

├── vercel.json     # Vercel 정적 호스팅 설정 (보안 헤더 등)

## eBay 링크 연결 ★

구매 CTA 버튼은 현재 **eBay 홈(`https://www.ebay.com`)** 으로 연결되어 있습니다(임시 placeholder, 새 탭에서 열림).
실제 리스팅이 준비되면 `index.html` 안의 **`https://www.ebay.com` 3곳**을 본인 리스팅 URL로 바꾸면 됩니다.

CTA 위치 (3곳):
- 히어로 — `지금 내 빛 두르기 →`
- 주문 섹션 — `결제하고 내 빛 받기 →`
- 마지막 섹션 — `지금 내 빛 두르기 →`

## 배포 (Vercel)

GitHub `growandrun/velloria` 저장소가 Vercel 프로젝트 `velloria`에 **git 연동**되어 있어,
`git push` 하면 **자동으로 재배포**됩니다. (`vercel --prod` 수동 실행 불필요)

## 콘텐츠 원칙 (중요)

근거 없는 과장·허위 표현은 넣지 않습니다. 제품 스톤은 **큐빅 지르코니아(CZ)** 입니다.
금지 예시: 고급 보석 표기, "3중 도금 / 변색 방지 / 평생 보증", "니켈프리", "당일 발송",
가짜 별점·후기 수, 거짓 마감 타이머 등. **실제 검증된 후기와 확인된 제품 사실만** 사용합니다.

확인된 사실: 큐빅 지르코니아(CZ) · 실버 진공 도금 · 폴드오버 잠금 · 폭 4mm · 길이 18 / 19cm · 약 12g · 남녀 공용.
